Scripts ScriptObject ScriptObjectWeapon

Base class of all weapon scripts, associated with a kWeapon actor. Weapon classes are defined in scripts/weapons.

Ammo is managed by kPlayer, which you can get a reference to with kWeapon::Owner().

Inherited Functions

void OnSpawn()

definitely gets called before the player can use the weapon, but I'm not sure exactly when

void OnTick()

called every tick, if it's the current weapon

Weapon Functions

void OnBeginFire()

called when the player presses the fire button (only if playing idle anim?)

void OnFire()

called every tick, if an attack animation is playing

void OnEndFire()

called when the player releases the fire button (only if playing attack anim?)

void OnRaise()

called at the end of swap-in anim

void OnLower()

called at the end of swap-out anim, when changing weapons

void OnHoldster()

called at the end of swap-out anim, when climbing
