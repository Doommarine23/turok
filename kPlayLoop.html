Methods

int Ticks()

number of ticks since the current map started

unlike kActor::GameTicks(), when the game's time scale is less than 100% (gamespeed < 1.0, or spirit mode is active), this will increment in proportion to the time scale (e.g., every 5 ticks when game speed is 1⁄5)

void TagActorForBossBar( kActor@ boss )

show boss health bar on the HUD, tracking boss's health

void RemoveBossActor()

remove boss health bar from the HUD

void CheckKeys()

void StartWarp( kActor@ a, int warpTID, int mapID )

teleport the player to the warp exit point with the given tag ID, on the map with the given map ID

a must be non-null, but otherwise doesn't seem to matter

void StartFreeWarp( kActor@ a, kVec3& dest, float yaw, int sector, int mapID )

teleport the player to an arbitrary point in the given sector, on the map with the given map ID, and facing the given direction

a must be non-null, but otherwise doesn't seem to matter

void HandlePlayerDeath( bool [unknown] = true )

called by map scripts at the end of custom player death cutscenes to trigger respawn or game over
