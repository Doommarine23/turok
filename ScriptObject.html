Scripts ScriptObject

This is the base class for all scripts that correspond to a kActor.

Script objects get a handle to their associated kActor in their constructor, which is typically stored in a member variable named self.

Methods

void OnTick()

gets called every tick

void OnSpawn()

gets called when the actor is spawned

flags and tag ID aren't initialized until OnBeginLevel() – possibly other properties, too

void OnBeginLevel()

gets called when the map starts, after OnSpawn()

only gets called for actors placed in the map file, not dynamically spawned actors

void OnPostBeginLevel()

gets called after OnBeginLevel()

this gets called for all actors – not just those placed in the map file – so you can think of it as a general "on-post-spawn"

void OnEndLevel()

void OnActivate()

void OnDeactivate()

void OnDamage( kActor@ instigator, kDictMem@ damageDef, const int damage )

void OnDeath( kActor@ killer, kDictMem@ damageDef )

gets called when taking damage, before the actor's health gets decremented

damageDef stores damage type properties

if the actor dies, OnDeath() is called before OnDamage()

OnDamage() doesn't get called for the player if he's invincible
